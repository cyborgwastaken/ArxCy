ArxCy Language Documentation

Welcome to the official language documentation for ArxCy. This document serves as a comprehensive reference for the syntax, data types, and features of the language.

1. Comments

ArxCy uses C-style single-line comments. Any text following a // on the same line is ignored by the transpiler.

// This is a single-line comment.
int x = 10; // This is an inline comment.

2. Data Types

ArxCy is a statically-typed language. It supports several primitive data types and single-dimensional arrays.

Primitive Types

Type Description Example Literal
intA 32-bit signed integer.42
fltA floating-point number.3.14
charA single character.'A'
boolA boolean value.true, false
strA string of characters."Hello"

Arrays

Arrays are fixed-size, ordered collections of elements of the same type.

  • Declaration: type[] array_name;
  • Initialization: type[] array_name = {value1, value2, ...};
// An array of integers
int[] numbers = {10, 20, 30};

// An array of booleans
bool[] flags = {true, false, true};

3. Variables

Variables must be declared with their type before they are used. They can be initialized at the time of declaration or assigned a value later.

Declaration & Initialization

// Declaration without initialization
int x;

// Declaration with initialization
str message = "Hello, ArxCy!";
int[] my_array = {1, 2, 3};

Assignment

The = operator is used to assign a new value to a variable.

int y = 10;
y = 20; // Re-assigning the value of y

int[] data = {1, 2, 3};
data[0] = 100; // Assigning a new value to an array element

4. Operators

ArxCy supports a standard set of arithmetic, comparison, and logical operators.

Category Operators
Arithmetic+ (Add), - (Subtract), * (Multiply), / (Divide), % (Modulo)
Comparison==, !=, <, >, <=, >=
Logical&& (AND), || (OR), ! (NOT)
Increment++ (Prefix and Postfix)
Decrement-- (Prefix and Postfix)
 
int result = (10 + 5) * 2; // result is 30
bool is_true = (result > 10) && !false; // is_true is true

5. Control Flow

if / else-if / else Statements

ArxCy uses an if-then-else structure for conditional logic. Code blocks must be enclosed in curly braces {}.

int score = 85;

if (score >= 90) then {
    print("Grade: A");
} else-if (score >= 80) then {
    print("Grade: B");
} else {
    print("Grade: C or lower");
}

for Loop

The for loop is ideal for iterating a known number of times.

// Prints numbers 0 through 4
for (int i = 0; i < 5; i++) {
    print(i);
}

while Loop

The while loop executes as long as its condition is true.

int counter = 3;
while (counter > 0) {
    print(counter);
    counter--; // or --counter
}

do-while Loop

The do-while loop is similar to while, but its body is guaranteed to execute at least once.

int x = 10;
do {
    print("This will print once.");
} while (x < 10);

6. Arrays In-Depth

Accessing Elements

Array elements are accessed using zero-based indexing with square brackets [].

int[] data = {10, 20, 30};
print(data[0]); // Prints 10
print(data[2]); // Prints 30

.length Property

You can get the number of elements in an array using the .length property. This is useful for writing safe loops.

int[] scores = {99, 87, 100, 91};

print("The number of scores is: " + scores.length); // Prints 4

// Loop through the array without hardcoding the size
for (int i = 0; i < scores.length; i++) {
    print(scores[i]);
}

7. Built-in Functions

print()

The print() function outputs a value to the console, followed by a newline.

Basic Usage:

print("Hello, World!");
print(123);

String Concatenation:

The print function has a special feature: it can concatenate a string literal with a single variable of another type using the + operator.

int age = 21;
print("My age is: " + age); // Correctly prints "My age is: 21"

flt pi = 3.14;
print("PI is approx: " + pi); // Correctly prints "PI is approx: 3.140000"
